Barbaros is a lightweight access control library designed for serverless and edge environments.

## Key Features

- Serverless-first: Optimized for stateless runtimes
- Simple API: Easy to understand and integrate
- Flexible: Works with various authentication methods
- Performance: Minimal overhead for quick access checks

## Use Cases

- API access control
- Feature flagging
- Role-based access control (RBAC)
- Multi-tenant applications
